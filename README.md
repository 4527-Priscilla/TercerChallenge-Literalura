<h2 align="center"> Proyecto Alura Latam One - Practicando Spring Boot</h2>

Este proyecto forma parte del curso Alura Latam One - Practicando Spring Boot. En este desafío se realiza una aplicación de consola desarrollada con Spring Boot, que permite a los usuarios interactuar con la API de Gutendex (un índice de libros del Proyecto Gutenberg) y persistir la información de los libros y autores consultados en una base de datos local.
